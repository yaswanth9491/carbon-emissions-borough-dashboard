# Borough-Level Carbon Emissions Dashboard

This project presents a Tableau dashboard analysing carbon dioxide emissions across boroughs.

## Dashboard Objectives
- Identify highest emitting boroughs
- Analyse emissions trends over time
- Understand sector-wise emission contributions
- Support sustainability and environmental analysis

## Tools Used
- Tableau Desktop
- Microsoft Excel
- GitHub

## Key Features
- KPI indicators for total emissions and highest emitting borough
- Interactive filters by year, sector, and borough
- Trend analysis using line charts
- Sector and borough comparisons

## Dataset
The dataset contains borough-level carbon emission data sourced from a public environmental dataset.

## How to Use
1. Download the Tableau workbook
2. Open in Tableau Desktop
3. Use filters to explore trends and insights
